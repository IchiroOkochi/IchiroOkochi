## Featured Project

<br/>

### 🌐 VaaniConnect

> *Bridging language barriers across India through machine learning.*

VaaniConnect is a translation system leveraging large language models to facilitate communication across Indian regional languages — a domain where robust, accessible tooling remains limited.

| **Focus** | LLM-powered translation for Indian languages |
